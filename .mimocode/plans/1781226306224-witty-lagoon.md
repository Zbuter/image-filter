# 分组功能 + 导出格式选择

## 目标
1. 图片分组：支持新建/重命名/删除分组，自定义快捷键
2. 分组标记：图片卡片角落显示分组名
3. 分组筛选：已选视图可按分组筛选
4. 导出格式：导出时弹窗选择 RAW/JPG/两者

## 数据结构

```typescript
// stores/app.ts
interface ImageGroup {
  id: string           // uuid
  name: string         // 分组名称
  shortcut: string     // 快捷键（如 "1", "2", "g"）
  paths: string[]      // 分组内的图片路径
}

// store 新增状态
const groups = ref<ImageGroup[]>([])     // 所有分组
const groupMap = ref<Map<string, string>>(new Map())  // path -> groupId
```

## 详细设计

### 1. Store — 分组状态与操作 (`stores/app.ts`)

```typescript
// 新增状态
const groups = ref<ImageGroup[]>([])
const groupMap = ref<Map<string, string>>(new Map()) // path -> groupId

// 分组操作
function createGroup(name: string, shortcut: string): ImageGroup
function deleteGroup(id: string)
function renameGroup(id: string, name: string)
function setGroupShortcut(id: string, shortcut: string)
function addToGroup(imagePath: string, groupId: string)
function removeFromGroup(imagePath: string, groupId: string)
function getGroupForImage(path: string): ImageGroup | null
function getGroupsForImage(path: string): ImageGroup[]

// 快捷键处理
function handleGroupShortcut(shortcut: string, currentPath: string)

// 持久化
function saveGroups()
function loadGroups()

// 导出格式
type ExportFormat = 'both' | 'raw' | 'jpg'
const exportFormat = ref<ExportFormat>('both')
```

### 2. 图片卡片分组标记 (`ImageGrid.vue`)

在 `.card-image` 区域左下角添加分组标签：
```html
<div v-if="getImageGroups(image.path).length > 0" class="group-tags">
  <span v-for="g in getImageGroups(image.path)" :key="g.id" class="group-tag"
        :style="{ background: getGroupColor(g.id) }">
    {{ g.name }}
  </span>
</div>
```

### 3. 右键菜单 (`ImageGrid.vue`)

现有菜单增加子菜单「添加到分组」：
```
标记为废片
标记为非废片
───────────
添加到分组 → 分组1 / 分组2 / 新建分组...
───────────
删除
```

### 4. 已选视图分组筛选 (`SelectedImagesView.vue`)

在顶部工具栏添加分组筛选下拉：
```html
<select v-model="filterGroupId" class="group-filter">
  <option value="">全部</option>
  <option v-for="g in store.groups" :key="g.id" :value="g.id">{{ g.name }}</option>
</select>
```

### 5. 预览页分组快捷键 (`ImagePreview.vue`)

在 handleKeydown 中添加：
```typescript
// 数字键 1-9 或自定义快捷键
default:
  if (store.isGroupShortcut(e.key)) {
    store.handleGroupShortcut(e.key, store.previewImage.path)
  }
```

### 6. 导出弹窗 (`ExportDialog.vue`)

在现有弹窗中添加格式选择：
```html
<div class="export-format">
  <label>导出格式：</label>
  <radio-group v-model="format">
    <radio value="both">RAW + JPG</radio>
    <radio value="raw">仅 RAW</radio>
    <radio value="jpg">仅 JPG</radio>
  </radio-group>
</div>
```

### 8. 持久化

分组数据存储到 Tauri app_data_dir：
- 文件: `groups.json`
- 加载时机: `autoLoadWasteModel()` 时同时加载
- 保存时机: 每次分组操作后自动保存
