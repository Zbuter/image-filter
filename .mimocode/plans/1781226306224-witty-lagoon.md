# 废片列表快速定位方案

## 目标
废片列表支持：1) 点击定位右侧图片；2) Hover 显示缩略图预览。

## 现有机制
- `highlightedWastePath` — hover 时高亮右侧卡片（ImageGrid 已有 `.waste-highlight` 样式）
- `previewFromAi` — 点击打开 ImagePreview（但不是定位到网格）
- `getImageUrl()` — 缩略图加载（RAW 用 cache，普通图用 convertFileSrc）
- `rawPreviewCache` — RAW 预览缓存

## 方案

### 1. Hover 显示缩略图（AiPanel.vue）
在废片卡片左侧添加 48x48 缩略图，复用 `getImageUrl` 逻辑。

**修改文件**: `src/components AiPanel.vue`

```vue
<!-- 废片卡片加入缩略图 -->
<div class="waste-card">
  <img class="waste-thumb" :src="getThumbUrl(item.path)" />
  <div class="waste-info">
    <div class="waste-name">...</div>
    ...
  </div>
</div>
```

添加 `getThumbUrl(path)` 函数，对 RAW 文件调用 `store.getRawPreview()`，普通文件用 `convertFileSrc()`。

### 2. 点击定位到网格（App.vue + ImageGrid.vue）
点击废片项 → 滚动到对应图片卡片并高亮。

**修改文件**:
- `src/App.vue` — 新增 `scrollToImage(path)` 函数
- `src/components/ImageGrid.vue` — 新增 `scrollToPath(path)` 方法，通过 `ref` 暴露给父组件

实现：
1. App.vue 监听 `@scroll-to` 事件
2. ImageGrid 接收 `scrollToPath` prop（或通过 ref 调用）
3. 滚动逻辑：在 `filteredImages` 中找到目标索引 → RecycleScroller 的 `scrollToItem(index)` → 触发 highlight

### 3. 具体修改

#### AiPanel.vue
- 添加 `waste-thumb` 样式（48x48, object-fit: cover, border-radius）
- 添加 `getThumbUrl(path)` — 从 store 获取缩略图
- 卡片布局改为 flex row（缩略图 + 信息）

#### App.vue
- `previewFromAi` 改为同时调用 `scrollToImage`
- 新增 `scrollToImage(path)` — 调用 ImageGrid 的滚动方法

#### ImageGrid.vue
- 暴露 `scrollToPath(path)` 方法（defineExpose）
- 内部实现：找到 index → `recycleScroller.scrollToItem(index)` → 设置 highlight

#### stores/app.ts
- 新增 `scrollToTarget` ref，供 ImageGrid watch

## 验证
1. `npx tauri dev` 启动
2. 打开图片目录 → 点击分析
3. hover 废片项 → 确认缩略图显示
4. 点击废片项 → 确认右侧网格滚动到对应图片并高亮
5. `npx vue-tsc --noEmit` 确认无 TS 错误
